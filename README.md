# AI-Powered Customer Churn Prediction

## Project Overview
This project uses Machine Learning to predict customer churn behavior using telecom customer data.

The system analyzes customer details and predicts whether a customer is likely to leave the service.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

---

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Prediction
- AI-Based Churn Detection
- Streamlit Web Application

---

## Machine Learning Model
- Random Forest Classifier

- Accuracy: 78.96%
